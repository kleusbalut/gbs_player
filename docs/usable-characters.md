# 使用可能文字リスト

GBS Player の曲名、カスタムタイトル、カスタム作者名は独自の1バイト文字コードで保存されます。
対応していない文字はビルド時またはSAV保存時にスキップされます。

## 使用できる文字

### ASCII

半角ASCIIの `0x20` から `0x7E` までを使用できます。

```text
 !"#$%&'()*+,-./0123456789:;<=>?
@ABCDEFGHIJKLMNOPQRSTUVWXYZ[\]^_
`abcdefghijklmnopqrstuvwxyz{|}~
```


### 記号

```text
ー ゛ ゜
```

長音は全角 `ー` と半角 `ｰ` を入力できます。
濁点は `゛`、結合濁点 `゙`、半角 `ﾞ` を入力できます。
半濁点は `゜`、結合半濁点 `゚`、半角 `ﾟ` を入力できます。

## 濁点・半濁点付き文字

濁点・半濁点付きのかなは、内部では「基底文字 + 記号」の2文字として保存されます。

例:

```text
が = か + ゛
ぱ = は + ゜
ヴ = ウ + ゛
```

### 濁点付き

```text
がぎぐげご
ざじずぜぞ
だぢづでど
ばびぶべぼ
ゔ
ガギグゲゴ
ザジズゼゾ
ダヂヅデド
バビブベボ
ヴ
```

### 半濁点付き

```text
ぱぴぷぺぽ
パピプペポ
```

## 文字数の注意

- 曲名、カスタムタイトル、カスタム作者名は最大31バイトです。
- ASCII、清音のひらがな/カタカナ、小書き文字、長音は1バイトです。
- 濁点・半濁点付き文字は2バイトです。
- 31バイトを超えた部分は切り捨てられます。
- 漢字、全角英数字、全角記号、絵文字、改行などは使用できません。

## 内部エンコード

| 範囲 | 内容 |
|---|---|
| `0x00` | null終端 |
| `0x01` | ゛ |
| `0x02` | ゜ |
| `0x03` | ー |
| `0x04`-`0x0C` | 小書きひらがな |
| `0x0D`-`0x15` | 小書きカタカナ |
| `0x20`-`0x7E` | ASCII |
| `0x80`-`0xAD` | ひらがな |
| `0xB0`-`0xDD` | カタカナ |
