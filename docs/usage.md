## 操作方法・オプション画面項目説明

[English](usage.en.md)

### 曲リストモード

<img src="../images/2026-05-10 234235.png" width=300px>

- `A`: 再生 / 一時停止
- `B`: 停止
- `↑ / ↓`: カーソル移動
- `← / →`: 前の曲 / 次の曲へ移動
  - 再生中のみ有効
- `SELECT` プレイリストモードへ切替
  - プレイリストが空の場合は切替しません
- `START`: オプション画面を開く

曲リストモードでは全曲一覧を表示します。プレイリストが空の状態で再生を始めた場合は、この曲順に従って自動曲送りします。

### プレイリストモード

<img src="../images/2026-05-10 234254.png" width=300px>

- `A`: 再生 / 一時停止
  - 曲移動中は確定
- `B`: 停止
  - 曲移動中はキャンセル
- `↑ / ↓`: カーソル移動
- `← / →`: 前の曲 / 次の曲へ移動
  - 再生中のみ有効
- `SELECT`:
  - [短押し] 曲リストモードへ切替
    - 曲移動中は移動を解除
  - [長押し] 曲をつかむ / 置く
- `START`: オプション画面を開く

プレイリストモードで再生を始めた場合は、プレイリスト順で自動曲送りします。並べ替え中は、つかんだ曲を別の位置へ差し込む形で順序を変更できます。

### オプション画面

<img src="../images/2026-05-10 234307.png" width=300px>

- `↑ / ↓`: 項目選択
- `A / ← / →`: 選択項目の値を変更
- `B`: オプション画面を閉じる

設定項目:

- `ADD PLAYLIST`: 選択中の曲をプレイリストへ追加
- `DEL PLAYLIST`: 選択中の曲をプレイリストから削除
- `Move`: 曲の並べ替えを開始 (プレイリストモードのみ)
- `Time`: 曲ごとの再生時間設定
- `Fade`: フェードアウト時間設定
  - 指定した秒数をかけて緩やかにフェードアウトします。
- `Sil`: 無音検出時間設定
  - この時間無音が検出されると、再生時間設定を無視して次の曲へ曲送りします。
- `RPT`: リピートモード (`---` / `ONE` / `ALL`)
- `Out`: 音声出力モード (`STEREO` / `MONO`)

プレイリスト内容と各種設定は SRAM に保存されるため、次回起動時にも引き継がれます。
